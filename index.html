No Applicable Code
Field Reference

Purpose of Code 298

Identify conditions without existing codes
Code 298 is used to identify infraction conditions that don't have an existing code but still need some sort of repair or follow-up action. It serves as a catch-all for undocumented conditions.

Not for CIP Issues

Critical exclusion to remember
Do not use Code 298 for CIP (Cathodic Inspection Program) issues. CIP conditions have their own dedicated codes and must be documented through the proper CIP reporting channels.

Emergency Protocol

Notify supervisor & stand by
For emergencies, you must notify the electric construction supervisor and stand by until the supervisor arrives on site. Do not leave the location until you are relieved.

Documentation Required

Detailed comments in MDT iPad
Enter a detailed description of the condition in the comments section of your MDT iPad. Be thorough — since there's no standard code, your notes are the primary record.
